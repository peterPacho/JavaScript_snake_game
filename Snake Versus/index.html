Display small box which contain some informations about game performance.
There is some most important values that affect game performance, rules and behavior. Setting them to some abstract values may broke game or freeze browser. Don't change them when game is running (not in menu or wasted screen). After refresh everything back to normal. If you wish so you can play with them. Remember to read description to know what every of these values do and what range they have (for example for some of them setting it to 0 will broke game). Setting some of them to something bad will need refresh of this page.
It's just efficiency limit. It protect game from shuttering and on this limit depends maximal number of "food" squares generated. Accepted values - almost everything less than 100. Make it lower than 90. Low values = more food but higher chance to slowdown.
Starting length of snake. Too much and browser will stop working. Accepted values - from 0 to some large number but don't make it too much. Let's say that 200 should be fine. Anything larger - just make it 0. Larger value = larger snake. Make it 0 to make endless snake
Size of every rectangle in game (so it affect on both snakes and food). Accepted values - from 1 to 600. When you make snake larger game seems to be faster so try playing with both this and "updated per sec" value. When you make it more than 200 game will probably not be playable.
Number of game updates per second. Affect game FPS (or [here] refreshing rate of canvas). Affect on snake speed. Every update change snake position by 1. Accepted values - from 1 to infinity (there border is computer performance, but values higher than 500 probably would not work).
Part of snake which can be eaten by other snake (this different-colored part). Accepted values - higher than 0, up to 1 (0>value>=1). It's like percentage value of snake so if you wish 50% will be eatable, make it 0.50. 1 means you can't eat anything. 0 will broke game
How valuable food is or how many squares will be added to snake after eating white chunk or how fast snake grow. Accepted values - from 0 to some large number. 0 means snake will not grow. Too high and game will slow down.
How many food squares are generated each update. Accepted values - from 0 to some large number. Too high and game will slow down. 0 means that no food is generated.
Resolution of game window. Accepted values - from 500 to infinity? Best to just press "Set Max" button.
